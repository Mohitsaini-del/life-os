# 🚀 Life OS

A full-stack productivity platform to manage goals, habits, notes and get AI-powered daily planning.

## ✨ Features

- 🔐 Authentication (NextAuth Credentials)
- 🎯 Goal Management
- 🔥 Habit Tracking + Streaks
- 🧠 Notes / Second Brain
- 🤖 AI Life Assistant
- 📊 Productivity Dashboard
- 🌙 Modern Dark UI

## 🛠 Tech Stack

Frontend:
- Next.js 16
- React
- TypeScript
- Tailwind CSS

Backend:
- Next.js API Routes
- Prisma ORM
- PostgreSQL (Neon)

Authentication:
- NextAuth.js
